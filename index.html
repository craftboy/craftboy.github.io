小表匠的储藏室

2014·春·一

View My GitHub Profile

我的忧郁气质是否继承自我妈妈？我听到街道上广场舞的音乐，想到那群跟妈妈年纪一般的阿姨们随着音乐扭动。我想她们全都面带笑容、兴高采烈，享受经济飞速发展的中国社会带来的富足快乐。而我的妈妈，她此刻正搬了一个四角的红木椅，坐在阳台上一动不动。她总是这样耽于思考。那些被身体蓄积的能量全都供给到如机械工业般沉重运作的脑力活动上。她也许在琢磨明天该买些什么菜；也许在想这潮湿阴冷的天气会持续多久；但她也许跟我一样，总在思考，这虚无的人生存在的意义。
